Step 1: Chạy code server
   (*) cd code-server
   (*) npm install
   (*) npm run serve

Step 2: Chạy code client
    (*) cd code-client/client
    (*) npm install
    (*) npm run start

    (*) Khi đã có tài khoản người có thể đăng bài post và like ,bình luận và xóa bài post của chính mình
